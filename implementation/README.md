# Optional Implementation

## Overview

This repository primarily focuses on the High-Level Design (HLD) of a scalable News Feed System.

A production implementation would include the following microservices:

- User Service
- Authentication Service
- Post Service
- Feed Service
- Search Service
- Notification Service
- Media Service

### Suggested Tech Stack

- Java Spring Boot
- PostgreSQL
- Cassandra
- Redis
- Apache Kafka
- Docker
- Kubernetes

Future implementation can include:

- REST API implementation
